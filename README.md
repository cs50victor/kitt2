# KITT V2

## Prerequisitise
**Install Vulkan**
[Vulkan Getting Started (Mac)](https://vulkan.lunarg.com/doc/sdk/1.3.268.1/mac/getting_started.html)

3. `ngrok http http:://localhost:4000` ( in a different terminal )
